go to wonmo cyber school

wonmo cyber school은 Micro Service Architecture에 영감을 받아 만든 웹 어플리케이션이에요.
유저 로그인과 인증, 게시물 관리, 이미지 서비스, 웹 UI 기능을 각각 담당하는 4개의 마이크로 서비스로 이루어진 프로젝트이지요.
방구석 홈서버(우분투 서버 20)에 호스팅 중입니다.
이 실험을 통해 MSA의 장단점을 파악할 수 있었지요.

wonmo cyber school is a web application inspired by the Micro Service Architecture.
Each feature like User authentication, API for Posts, image serving, web UI is divided as an independent app.
Hosting in my home server(ubuntu server 20)
Through this, I had learned how MSA works.
